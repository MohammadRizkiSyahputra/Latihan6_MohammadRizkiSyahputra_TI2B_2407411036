6. CSS Float

Float memungkinkan elemen "mengambang" ke kiri atau kanan dari container-nya.

Float: left

Teks ini akan mengalir di sekitar elemen yang mengambang ke kiri. Float sering digunakan untuk layout dan penempatan gambar.

Color: rgb(0, 0, 255) (RGB)

Color: rgba(255, 0, 0, 0.5) (RGBA dengan transparansi)

9. CSS Text

text-decoration: underline
